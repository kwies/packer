Name: centos-7upd-x86_64
Description: 
 - CentOS 7 server, 64 bit.
 - Build for virtualbox, guest extensions installed.
 - Updates installed.
 - Built for vagrant
 - Locales: US (Language,Timezone,Mirror)
Notes:
 - Firewall and SELinux are deactivated



IMPORTANT NOTES
===============

to run CentOS 7 with vagrant and virtualbox be ensure
 - Vagrant 1.6.5+ is installed
 - VirtualBox 4.3.14+ is installed
 - vagrant-plugin "vagrant-vbguest" is installed:
      vagrant plugin install vagrant-vbguest
   Add in Vagrantfile following line:
      config.vbguest.iso_path = "http://download.virtualbox.org/virtualbox/4.3.14/VBoxGuestAdditions_4.3.14.iso"
